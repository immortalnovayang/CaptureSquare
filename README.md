# CaptureSquare  
截圖做ICON  

功能說明  
擷取畫面補透明成矩形圖片(透明PNG)  

需求環境  
JDK 1.8  

操作方式  
0.執行目錄底下的CaptureSquare.jar (點兩下)  
1.按下進入擷取模式按鈕，就可開始擷取圖片  
2.按住要截取的圖片起始點  
4.拖曳到要截取的圖片終點放開，即可產生png圖片在目錄下(拖曳途中會有紅線標示範圍)  
4.如想取消(退出擷取模式)，就點擊右鍵
